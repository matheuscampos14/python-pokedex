# Python Pokédex

A command-line Pokédex application built with Python using the PokéAPI REST API.

## Features

- Search Pokémon by name
- Display Pokémon information
- Show types and abilities
- Display sprites and cries
- Handle invalid Pokémon names

## Technologies

- Python
- Requests
- REST API
- JSON
